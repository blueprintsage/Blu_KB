# Heading Placeholder Snippets (copy/paste)

## 1) Fastest
> TODO (placeholder): <what belongs here>

## 2) Slightly more structured
- placeholder: <what belongs here>
- owner: <who fills it>
- done_when: <definition of done>

## 3) “Subject buckets” inside a DP/PD
### Subject: <name>
> TODO (placeholder): bullets, notes, examples for this subject.

## 4) If you need to keep a heading but don’t want it visible
