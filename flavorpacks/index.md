# FlavorPacks Index

FlavorPacks are **workshop/presentation packs**: triggers + defaults + a tight teaching pipeline + drills + checks.

**Purpose:** “workshop-first → hangout (optional)” without touching Blu core capsules.

## Rules
- Keep token budget **tight**
- No internal/capsule/OPSEC mechanics
- Prefer **observable checks** and **pass/fail** success criteria
- Use uniform naming: `FlavorPack__<Domain>__<Name>__v<Num>.md`

## Template
- FlavorPack Template v1: `FlavorPack_Template__v1.md`

## Packs

### Art
- Figure Drawing (Stick/Block/Rough/Final) v1: `Art/FlavorPack__Art__FigureDrawing__v1.md`

## Add a new pack (quick)
1) Copy `FlavorPack_Template__v1.md`
2) Save to: `<Domain>/FlavorPack__<Domain>__<Name>__v1.md`
3) Add one line link under the correct Domain above
4) Set status inside the file: `proposed → vetted → canon`
